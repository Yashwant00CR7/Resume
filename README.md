# Interview Prep & Resume Repository

This repository contains comprehensive interview preparation materials tailored for roles in Agentic AI, Data Engineering, and ML Engineering.

## Contents
- `questions.txt`: A bank of 120 technical and behavioral interview questions.
- `answers.txt`: Detailed answer key for all 120 questions.
- `resume.tex`: LaTeX source code for the professional resume.

1. Resume & Project Deep Dive
2. Machine Learning & Deep Learning Fundamentals
3. Data Engineering
4. System Design
5. Python & Coding
6. Scenario-Based / Real-World Problem Solving
7. HR / Behavioral Questions
